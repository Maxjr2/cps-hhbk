# cps-hhbk

This repo comtains the code and documentation for the block 2 cps project.

## Project Structure

- `sensors/`: Contains the functions to get data from the defined sensors.
- `actors/`: Contains the functions to control the defined actors.
- `utils/`: Contains utility functions used across the project.
